Meta Bee API

指定したURLのメタ情報（サイト名、OGP画像、SNSアカウント、ドメイン取得日など）を取得するAPIです。

Try it now

Meta Bee APIとは

指定したURLにアクセスし、メタ情報やOGP画像、タイトル、ドメインの取得日、サイト内の文字列を解析し、運営元やTwitter・YouTube・GitHubアカウントなどを抽出します。

使い方

APIエンドポイントに調べたいURLをURLパラメータとして与えることで、JSONで結果を得られる。

  • cURL
  • Ruby 
                                            
curl -X GET https://api.appp.la?url=https://www.appp.la \
  -H "Accept: application/json"
                                            
                                        
                                            
require 'faraday'
api_url = 'https://api.appp.la'
url = 'https://www.appp.la'
res = Faraday.get api_url, { url: url }
puts res.body

データは保存しない

ユーザーから与えられたURLパラメータや結果は保存していない。与えられたパラメータに対してその都度、結果を返している。
